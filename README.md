# Mini CRM de Contactos

Pequeño CRM para registrar contactos (clientes) y su estado, desarrollado con Next.js, React y Prisma. Permite agregar, editar, eliminar, filtrar contactos, ver historial de cambios y exportar la lista a CSV.

---

## Tecnologías utilizadas

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Next.js API Routes, Node.js
- **Base de datos:** Prisma + PostgreSQL (Supabase)
- **Gestión de versiones:** Git, GitHub
- **Despliegue:** Vercel
